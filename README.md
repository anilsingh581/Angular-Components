## How to Run
- Clone this repository:
  ```bash
  git clone https://github.com/yourusername/angular-interview-examples.git
  ```
- Navigate into the folder:
  ```bash
  cd angular-interview-examples
  ```
- Install dependencies:
  ```bash
  npm install
  ```
- Run the Angular application:
  ```bash
  ng serve
  ```
